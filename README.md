# 프로젝트 개요
본 미션은 코드를 작성하기 전, 개발의 토대가 되는 **'환경 세팅'**과 **'재현 가능한 인프라 구축'**능력을 기르는 것을 목표로 합니다.

1. 핵심 도구의 숙련 및 기반 구축
  * Terminal (Linux CLI): 작업 디렉토리 관리 및 권한 설정을 통한 시스템 제어 능력 함양
  * Docker (Container): 실행 환경의 격리와 재현성을 확보하여 환경 의존성 문제 해결
  * Git/GitHub: 효율적인 버전 관리 및 팀 협업을 위한 소스코드 관리 체계 이해
2. 구조적 원리 이해 및 실무 적용
  * Dockerfile 기반 컨테이너화: 웹 서버를 직접 이미지화하며 인프라의 코드화 경험
  * 포트 매핑(Port Mapping): 네트워크 설정을 통한 외부 접속 및 서비스 노출 원리 파악
  * 데이터 영속성 검증: 바인드 마운트와 볼륨을 활용한 컨테이너 데이터 관리 방식 습득
